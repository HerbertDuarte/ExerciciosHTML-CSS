Reproduzindo audios

usando a tag SOURCE dentro da tag AUDIO e o parâmetro CONTROLS para habilitar o player

outros parâmetros

  • autoplay - iniciar a musica junto ao site (em alguns navegadores não funciona)
  • preload
    • auto - só carrega o site depois de carregar o audio(não é recomendado)
    • metadata - só carrega informações básicas do arquivo(nome, duração)
    • none - não carrega nada
  • loop - após terminar de reproduzir, o áudio vai recomeçar até que o usuário o pare

IMPORTANTE: Para melhor compatibilidade com navegadores é recomendado que faça o upload do arquivo nos 3 formatos, mp3, ogg e wav. Dessa forma, aumentam-se as chances de que o navegador seja compatível. É permitido e necessário por os 3 arquivos na mesma tag AUDIO, dessa forma o navegador vai testar os 3, na ordem colocada, e o que funcionar ele irá gerar no site.

obs: arquivos .wav são MUITO pesados e não são recomendados para o site
