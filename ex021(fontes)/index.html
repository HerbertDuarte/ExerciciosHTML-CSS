Fontes externas

Usa-se no arquivo de estilo do css o objetivo

    @font-face {
      font-family:'cafe';
      src: url('Louis-George-Cafe.ttf') format(truetype), url('Louis-George-cafe.ttf') format(opentype);
      font-weight:normal;
      font-style:normal;
    }

  • truetype(ttf)
  • embedded-opentype
  • truetype-aat (apple advanced typography)
