Pseudoclasses (:)

Hover : quando o mouse esiver em cima do objeto

Active : quando o objeto estiver ativo

Checked : quando o objeto estiver marcado (checkbox, radiobutton)

Visited : quando o usuário já visitou um link de TAG A

Obs: pode-se usar o > para se referir às tags filhas

PseudoElementos (::)

After - tudo que acontece depois do objeto

Before - tudo que acontece antes do objeto

Usando o content com e o conteudo entre aspas simples, é possivel inserir textos e talvez midias nesses pseudoelementos
