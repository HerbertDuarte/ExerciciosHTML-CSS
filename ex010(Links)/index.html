usa-se para configurar um link de download, a tag A e os parâmetros download, href e type

para saber o type correto para o arquivo, podemos pesquisar pelo tipo do arquivo usando F3 no site iana.org

obs: no caso do arquivo .png, usando o método acima o download é feito em .img. Para resolver retira-se o parâmetro type e é deixado o parâmetro download vazio sem o igual (=) e sem as aspas ("")
